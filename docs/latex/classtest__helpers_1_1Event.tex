\doxysubsection{Detailed Description}
Represents a MIDI event. 

The \doxylink{classtest__helpers_1_1Event}{Event} class encapsulates data related to MIDI events. This includes the event ID, whether the event has been handled, and potentially other MIDI-\/specific information such as the type of event (e.\+g., Note On, Note Off), the channel number, and additional data bytes (e.\+g., note number, velocity).
