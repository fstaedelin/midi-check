\doxysubsection{Detailed Description}
Mixin class providing utility functions for the Logger class. 

The \doxylink{classjarvis__utilities_1_1JarvisUtilitiesMixin}{Jarvis\+Utilities\+Mixin} class extends the Jarvis\+Printing\+Mixin class and includes several utility functions that manage logging contexts, handle automatic naming of contexts and tests, and trigger messages based on the outcomes of tests. These utilities are designed to work seamlessly with the logging system in \doxylink{namespaceJARVIS}{JARVIS}.

\begin{DoxyVerb}Mixin class for various utility functions used by the Logger class.
\end{DoxyVerb}
