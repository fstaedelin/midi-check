\doxysubsection{Detailed Description}
Main class for the \doxylink{classjarvis_1_1JARVIS}{JARVIS} logging system. 

The \doxylink{classjarvis_1_1JARVIS}{JARVIS} class extends Jarvis\+Utilities\+Mixin, incorporating a variety of logging and utility functions. This class provides methods for logging messages at different levels, navigating between contexts, adding and triggering tests, and generating callback messages. The class is designed to be the central point of interaction for the \doxylink{classjarvis_1_1JARVIS}{JARVIS} logging system.
