Detailed Description

Mixin class providing utility functions for the Logger class.

The MidiCheckUtilitiesMixin class extends the MidiCheckPrintingMixin class and includes several utility functions that manage logging contexts, handle automatic naming of contexts and tests, and trigger messages based on the outcomes of tests. These utilities are designed to work seamlessly with the logging system in MIDI_CHECK.

Mixin class for various utility functions used by the Logger class.

Automatically generates a name for a new context to avoid conflicts.
