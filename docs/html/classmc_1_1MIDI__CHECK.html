Detailed Description

Main class for the MIDI_CHECK logging system.

The MIDI_CHECK class extends MidiCheckUtilitiesMixin, incorporating a variety of logging and utility functions. This class provides methods for logging messages at different levels, navigating between contexts, adding and triggering tests, and generating callback messages. The class is designed to be the central point of interaction for the MIDI_CHECK logging system.

Add a new test to the current context.

This method adds a new test to the current context within the logging hierarchy. The test is defined by a test function, expected result, and optional callback functions for handling success and failure. The test is automatically named to avoid conflicts, and the method returns the test object.
